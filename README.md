# Xmuse EEG Paper Sharing Platform

## 📚 仓库简介

本仓库专注于**EEG（脑电图）**领域相关研究论文、开源代码及科研资料的整理与共享，旨在搭建一个开放、协作、共赢的脑机接口（BCI）学习交流平台。

仓库面向学生、科研人员及脑机接口技术爱好者，致力于汇聚优质EEG科研资源，方便大家快速查阅前沿文献、复现实验代码、交流研究经验，共同推进EEG信号处理、分类及应用领域的学习与创新。

***所有资源仅用于学术研究，禁止任何商业用途。***

## 📂 仓库目录说明

### 1\. 脑电分类论文

收录EEG信号分类相关高质量论文与研究资料，覆盖各类主流、新型信号分类算法、脑电特征提取技术，以及不同场景下的EEG分类应用研究，为信号算法研究提供全面的文献支撑。

### 2\. 脑电设备对比论文

汇总各类EEG脑电设备的性能对比、参数测评、适配场景相关研究文献，帮助科研人员与开发者快速筛选适配自身实验、开发需求的脑电采集设备，降低设备选型成本。

### 3\. 脑电分类代码

开源多款主流EEG信号分类算法完整代码，包含 **EmT\-main、PGCN\-main、LSTM\-main、PR\-RL\-main** 等项目。

- 每个代码项目均配备独立README文档，提供详细的环境配置、运行教程及使用说明，开箱即用；

- **LSTM\-main** 内置完整的EEG信号二分类示例代码，专为初学者和入门科研人员设计，可快速上手脑电信号预处理、特征建模与分类任务。

### 4\. Xmuse相关论文汇总

系统性整理**2015年至今**基于Muse便携式脑电设备的国内外研究文献，聚焦认知科学、情绪识别、冥想监测、临床医疗、人机交互等核心应用场景。

**文献统计（更新至2026\.07\.02）**：总计收录**216篇** 学术论文

- EI检索论文：107篇

- SCI检索论文：83篇

- 中科院2区及以上顶刊论文：28篇

## 🤝 开源贡献

本仓库为开源协作项目，欢迎所有爱好者参与共建，任何形式的优质贡献都十分欢迎，主要贡献方式包括：

- **新增文献/代码**：发现优质EEG相关论文、开源算法代码，可通过 Pull Request 提交至对应目录；

- **修正勘误**：若发现文档、文献信息、代码存在错误或疏漏，欢迎提交修正反馈；

- **经验分享**：可在 Issues 板块分享你的实验心得、代码调试经验、研究思路及创新想法。
